# Spatial and featural cue weighting in children’s developing object representations

This repository contains data and code for the paper **“Spatial and featural cue weighting in children’s developing object representations.”**  

All figures can be recreated by running the Jupyter notebook `cue_weighting_figures.ipynb`. 

## Results

Contains csvs with the results of the infant preschool experiments and models.

## Modeling
Contains scripts for filtering MS-COCO, pre-training the models for object individuation or classification, and evaluating the models on the cue-weighting task.
